SAP Fiori
effizient einführen

Lernen Sie mit der SAP Fiori apps reference library, der SAP Fiori apps rapid-deployment solution (RDS), SAP Task Lists (STC01) und App Activation die Werkzeuge für eine effiziente SAP Fiori Einführung kennen. Wie kommt man schnell von null auf SAP Fiori. Erfahren Sie auch wie SAP Gateway Services im Backend erweitert und dann in der SAP Web IDE genutzt werden können.
