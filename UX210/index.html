Tips and Tricks for an Efficient
Implementation of SAP Fiori

UX210, SAP TechEd 2016, Barcelona

For an efficient implementation of SAP Fiori, SAP provides different tools like the SAP Fiori apps reference library, SAP Fiori apps rapid-deployment solution (RDS), and more. Learn in this session the best practice for getting from zero to SAP Fiori in the most efficient way. Utilize task lists and the app activation tool. Extend SAP Gateway technology in the backend and the SAP Fiori app in SAP Web IDE. Join this presentation with additional live demos.

SAP Cloud Connector

SAP Administrator

  • Encrypted connection to the SAP Business Technology Platform
  • Access control list for hosts and services
  • Single Sign On using the Cloud authentication using Principle Propagation
  • Protocols (HCP to on-premise): HTTP, HTTPS, RFC
  • Protocols (on-premise to HCP): JDBC, ODBC
